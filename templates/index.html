Bem-vindo ao Umbrella Remember!

Este projeto nasceu de um simples script para avisar sobre chuva, e evoluiu para uma plataforma com interface gráfica intuitiva!

Como funciona?

  1. Cadastro: Informe seu nome, e-mail e cidade para começar a receber notificações personalizadas.
  2. Consulta diária: Todos os dias, às 6h da manhã, o sistema verifica a previsão do tempo para sua cidade.
  3. Notificação: Se houver previsão de chuva, você recebe um e-mail avisando para levar o guarda-chuva. Se não, aproveite o dia de sol!

Assim, você nunca mais será pego de surpresa pelo tempo!

Tecnologias Utilizadas

  • Python
  • HTML5
  • CSS3
  • SMTP/E-mail
  • MySQL
  • OpenWeather API
